This is a game that aims to achieve a 3D sandbox environment where players can spawn in vegetation and animals and watch the animals show suitable behaviour to their species and surroundings.

To see the source code and be able to edit everything or see the inner workings of the game, clone this repository and open it in Unity.

# Features
* Procedurally generated terrain with biomes and vegetation using perlin noise.
* Foxes and grasshoppers with AI.
* The class for animal AI can easily be expanded and inherited to create AI for new species of animals.
* Spreading vegetation.
* The player can manually spawn in objects.
* Camera movement with different speed options.
* Speeding up or slowing down the the simulation.
* Save system to save and load games at any time from the pause menu.
* Configuration options such as changing the resolution and window mode.
* A custom made console in the user interace for displaying important events.

# Controls
WASD - Movement

Holding Shift - Increases move speed

Holding Ctrl - Decreases move speed

Mouse - Look around

E - Bring up the spawn menu

ESC - Bring up the pause menu

Left click - Spawn the selected object
